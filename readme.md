# ⚡ MorseType – Morse Code Training App

MorseType is a modern, interactive Morse code training web application inspired by typing platforms like Monkeytype. It helps users learn and practice Morse code through real-time input, speed tracking, and accuracy feedback.

---

## 🚀 Features

* ⌨️ **Tap Input System**

  * Short press = Dot (.)
  * Long press = Dash (-)
  * Right click = End letter

* ⚡ **Real-Time Feedback**

  * Current letter cursor
  * Correct (green) / Wrong (red) highlighting
  * Live typing experience

* 📊 **Performance Stats**

  * Speed (CPM)
  * Accuracy (%)
  * Timer

* 🎯 **Custom Practice Modes**

  * Time-based mode
  * Word-based mode

* ⚙️ **Options**

  * Toggle numbers
  * Toggle punctuation
  * Adjustable time (15 / 30 / 60 / 120 sec)

* 🔤 **Dynamic Text Generation**

  * Random words generated based on selected options

* 📱 **Responsive UI**

  * Works on desktop and mobile

---

## 🧠 How It Works

MorseType simulates real Morse communication:

* Tap quickly → Dot (.)
* Hold → Dash (-)
* Right click → Convert symbol into a character

Example:

```
... → S
--- → O
```
